# GitHub — first push

Use this folder as **`github.com/qbitOS/qbitos-erika`**.

## One-time setup

```bash
cd /Volumes/qbitOS/00.dev/qbitos-erika

git branch -M main
git remote add origin https://github.com/qbitOS/qbitos-erika.git
# or SSH: git@github.com:qbitOS/qbitos-erika.git

git add -A
git status
git commit -m "chore: qbitos-erika compliance + Pages (Erika / qbitos-astro)"
git push -u origin main
```
